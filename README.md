## 🧠 PromptForge — AI Prompt Simulator

An interactive AI Prompt Simulator Web App built using HTML, CSS, and JavaScript, designed to mimic AI responses based on different personas, evaluate prompt quality, and help users improve prompt engineering skills.

## 🚀 Features
* **Multi-Persona AI Simulation** – Switch between Developer, Creative Writer, and Cybersecurity personas.
* **Prompt Evaluation System** – Real-time feedback on clarity, specificity, and improvement tips.
* **AI Response Generation** – Simulated responses based on intent (code, explain, summarize).
* **Variations Mode** – Generate multiple AI responses for the same prompt.
* **Compare Mode** – Side-by-side comparison of different AI personas.
* **Prompt History** – Save and reuse previous prompts.
* **Quick Prompt Templates** – Predefined prompts for faster usage.
* **Customizable Settings** – Adjust creativity, response length, and tone.
* **Dark/Light Theme Toggle** – Dynamic UI theming.
* **Safety Filtering** – Blocks unsafe or harmful prompts.
* **Typing Animation UI** – Realistic AI chat experience.

## 🎮 How to Use
**Select Persona**
* Developer → Technical/code responses
* Creative Writer → Storytelling responses
* CyberSec → Security-focused responses

**Enter Prompt**
* Type your prompt in input box
* Press Enter or click Generate

**View Evaluation**
* Clarity level
* Specificity score
* Suggestions for improvement

**Compare Responses**
* Enable Compare Mode
* View responses side-by-side

**Use Templates**
* Click quick templates or sidebar templates

**Adjust Settings**
* Creativity (Low / Medium / High)
* Response Length (Short / Medium / Long)
* Tone (Formal / Casual / Friendly)

## 🎯 App Objective
PromptForge helps users:
* Learn prompt engineering
* Understand AI response behavior
* Compare different AI styles
* Improve clarity and specificity of prompts
* Simulate AI outputs without API cost
